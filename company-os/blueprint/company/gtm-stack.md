## Data Flow

```
Cold Outreach (email/text/LinkedIn) -> Discovery Call (qualify) -> AI Audit (diagnose) -> Proposal (close) -> Retainer (deliver)

Chamber Seminar -> Sign-up for free audit -> AI Audit -> Proposal -> Retainer

Live Demo Meeting -> "Build it live" moment -> Proposal -> Retainer
```
