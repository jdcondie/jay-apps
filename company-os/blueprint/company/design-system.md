# Design System

## Colors

| Name | Hex | Usage |
|------|-----|-------|
| Navy | #1B2A4A | Primary text, headers, table headers (white text on navy background) |
| Orange | #E8541A | Accent, CTAs, highlights, section labels, divider lines |
| Light Background | #F5F0EB | Table alternating rows, page backgrounds |
| Gray Body Text | #4A4A4A | Body copy |
| White | #FFFFFF | Backgrounds, text on navy headers |

## Typography

- **Headings:** Plus Jakarta Sans, Bold/800 weight (web/JSX). Arial Bold (Word documents).
- **Body:** Plus Jakarta Sans, 400 weight, 15px (web). Arial, 11pt (Word documents).
- **Code:** System monospace

## Logo

Navy and orange wordmark. No logo file yet. Use text-based branding with the color system.

## Visual Style

Clean, professional, navy and orange brand system. Orange divider lines between major sections. Tables use navy header rows with white text and light background (#F5F0EB) for alternating rows. Bracketed [fill-in fields] for client customization in templates. No stock photos. Screenshots, diagrams, and data over illustrations.

## Document Standards

- All Word docs use the navy/orange brand system
- Orange divider lines between major sections
- Tables: navy header row with white text, light background alternating rows
- Heading hierarchy: H1 (navy, bold), H2 (navy, bold), H3 (orange or navy)
- No em dashes anywhere
- Plus Jakarta Sans for web/JSX, Arial for Word documents

## File Format Defaults

| Task | Format |
|------|--------|
| Client-facing documents (proposals, reports) | .docx |
| Interactive tools (calculators, dashboards) | .jsx (React) |
| Web pages | .html |
| Internal reference, playbooks, skills | .md |
| Presentations | .pptx |
