# Skills

Skills are .md files that teach Claude how to do specific tasks. Each one has frameworks, steps, examples, and anti-patterns.

## How to add skills

1. Copy the 5 starter skills from the gtm-skills/ folder in the starter kit
2. Customize each one for your business (Claude will interview you on first use)
3. Create your own skills for any workflow you repeat more than 3 times

## Creating a new skill

Tell Claude:

```
Look at how the existing skills are structured.
Help me create a new skill for [your workflow].
```

Claude will draft it based on the patterns in your existing skills.
